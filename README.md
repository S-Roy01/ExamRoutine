## 📌 About the Project

I needed something clean and professional to track exactly how much time I have left before each exam and how many days of gap I have in between them. Instead of constantly checking a PDF, I put together this single-file web tracker. 

Everything runs directly in the browser (HTML, CSS, and vanilla JavaScript). It automatically updates statuses based on the current time and uses local storage so my progress checkboxes don't wipe every time I refresh the page.

---

## 🚀 Key Features

*   ⏱️ **Live Countdown Timer:** Tracks exactly how many days, hours, minutes, and seconds are left until the exam starts (9:30 AM).
*   🚦 **Smart Status Updates:** Text highlights red when an exam is currently running, and the entire card automatically grays out once the exam finishes at 12:00 PM.
*   ☕ **Automated Gap Calculator:** Automatically calculates and displays the exact number of study gap days between subjects.
*   ✅ **Progress Tracking:** Interactive checkboxes for *Theory Read*, *Previous Question Solve*, and *Final Revision*.
*   💾 **Auto-Save:** Saves all checkbox data directly to the browser's local storage.
*   📱 **Fully Responsive:** Looks perfect on a desktop monitor, tablet, or mobile phone.

## 💻 How to Use Locally

If you want to run this on your own machine without using the live link:

1. Clone or download this repository.
2. Ensure the main file is named `index.html`.
3. Double-click `index.html` to open it in Chrome, Edge, or Brave.
4. Check your boxes as you study—the browser will remember them even if you close the tab.

## 🌐 Hosting (GitHub Pages)

This project is currently hosted for free using **GitHub Pages**. 

If you're looking to fork this and host your own version:
1. Go to your repository **Settings** (top tab) -> **Pages** (left sidebar).
2. Under "Build and deployment", set the Source to **Deploy from a branch**.
3. Select the **main** branch and click **Save**.
4. Give it a minute or two, and your live URL will be ready!
